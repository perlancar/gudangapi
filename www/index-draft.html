GudangAPI

GudangAPI menyediakan API berbagai Semua dapat diakses dengan mudah menggunakan JavaScript, PHP,

API apa saja yang tersedia?

Daftar negara, propinsi, dan kecamatan. Validasi berbagai data dan nomor: SIM, KTP, NPWP, dll. Parsing: nomor telepon, Algoritma: terbilang, terbilang menjadi angka, Finansial: mutasi internet banking (BCA, Mandiri, PayPal), dan masih banyak lagi.

Cara memakai

Gratis, tapi kebutuhan dedicated, trafik tinggi, atau algoritma spesifik mungkin tersedia secara berbayar.
