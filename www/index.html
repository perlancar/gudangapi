GudangAPI

Gudangnya API gratis bagi programer Indonesia. Dokumentasi detil menyusul, tapi beberapa demo singkat:

Kurs

$ curl 'http://api.gudangapi.com/v1/finance/currency/id/bca/get_bca_exchange_rate?from=USD&to=IDR'
[200, "OK",
 {"buy": 8425,
  "sell": 8675,
  "average": 8650,
  "spread": 250,
  "mtime_str": "20-Mei-2011 10:14 WIB",
 }
]

# baca petunjuk penggunaan API ini di: http://api.gudangapi.com/v1/finance/currency/id/bca/get_bca_exchange_rate;h

Terbilang

$ curl 'http://api.gudangapi.com/v1/number/phrase/id/terbilang?num=1023.4'
[200, "OK", "seribu dua puluh tiga koma empat"]

# baca petunjuk penggunaan API ini di: http://api.gudangapi.com/v1/number/phrase/id/terbilang;h

Dari Terbilang

$ curl 'http://api.gudangapi.com/v1/number/phrase/id/dari_terbilang?phrase=seribu+dua+puluh+tiga+koma+empat'
[200, "OK", 1023.4]

# baca petunjuk penggunaan API ini di: http://api.gudangapi.com/v1/number/phrase/id/dari_terbilang;h

Validasi NPWP pajak

$ curl 'http://api.gudangapi.com/v1/tax/id/npwp/validate_npwp?npwp=01.957.716.2-000.000'
[200, "OK"]

$ curl 'http://api.gudangapi.com/v1/tax/id/npwp/validate_npwp?npwp=01.957.716.7-000.000'
[400,"Invalid NPWP: wrong check digit"]

# baca petunjuk penggunaan API ini di: http://api.gudangapi.com/v1/tax/id/npwp/validate_npwp;h

Parsing NPWP pajak

$ curl 'http://api.gudangapi.com/v1/tax/id/npwp/parse_npwp?npwp=01.957.716.2-000.000'
[
 200,
 "OK",
 {"pretty":"01.957.716.2-000.000",
   "serial":"957716",
   "branch_code":"000",
   "local_tax_office_code":"000",
   "check_digit":"2",
   "taxpayer_code":"01",
   "normalized":"01.957.716.2-000.000"
 }
]
# baca petunjuk penggunaan API ini di: http://api.gudangapi.com/v1/tax/id/npwp/parse_npwp;h

Lainnya

Tersedia banyak API lain: validasi KTP, SIM, nomor telepon. Kurs mata uang, saham, konversi IP/negara, internet banking, kalender hari libur nasional. Daftar negara, propinsi, kecamatan. Kamus bahasa. Dan masih banyak lagi.
